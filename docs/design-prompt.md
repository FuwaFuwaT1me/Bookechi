> Готовый промпт. Скопируй блок ниже целиком в Claude. Он самодостаточен: содержит дизайн-токены,
> список экранов, состояния, mock-данные и формат результата.
> Палитра и heatmap взяты из реального кода проекта (`Color.kt`), так что прототип сразу попадает в стиль.

```

для трекинга чтения книг «Bookechi» в тёплом минималистичном стиле — уютный reading companion,
а НЕ generic Material app. Сделай единый современный UI с хорошим мобильным UX, единой
дизайн-системой и переиспользуемыми компонентами. Все тексты — на русском.

ФОРМАТ РЕЗУЛЬТАТА
- Один кликабельный прототип под мобильный экран (~390×844), все экраны навигируются между собой.
- Сначала собери дизайн-систему (токены + компоненты), затем экраны на её основе.
- Используй mock-данные (ниже). Покажи ВСЕ основные и пустые состояния.
- Собери так, будто это реальный продукт, готовый к дальнейшей разработке.

ВИЗУАЛЬНЫЙ ЯЗЫК
- Тёплая бежево-кремовая бумага, тёмно-коричневая («эспрессо») типографика, оранжевые акценты.
- Мягкие радиусы, много воздуха, одна чёткая визуальная иерархия на экран, без перегруза.
- Заголовки — тёплый serif (например Fraunces / Lora / Source Serif), тело — чистый sans (Inter).
- Тонкие тёплые границы вместо тяжёлых теней; обложки книг — мягкие скруглённые прямоугольники.

ПАЛИТРА = «Terracotta & Linen» — тёплый земляной минимализм. Акцент — ПРИГЛУШЁННАЯ терракота

10) SUCCESS / Прогресс сохранён:
```

## Как пользоваться

1. Скопируй блок в тройных кавычках выше в Claude (design).
2. Для итераций добавляй точечные правки: «сделай hero-card крупнее», «другой serif для заголовков»,
   «покажи только тёмную тему» и т.п.
3. Если решишь по открытым вопросам из `adr-001-redesign.md` (serif, прогноз дочитывания, оценка книги) —
   допиши соответствующие строки в раздел ЭКРАНЫ перед отправкой.
